This is pretty weird, but I got a call from a friend, and yes, indeed,
you can set up @INC via "use lib" using an enfironmental variable...

Is that obscure enough for a Friday morning post to this list? -laugh-

j

$ export J=/tmp
$ cat j.pl
use lib "$ENV{J}/lib";
print join "\n", @INC;
print "\n";

$ perl j.pl

Also the command line program 'mech-dump' rocks.  You can give it a URL
or a file and dumps forms (and optional links and images).  It's great
when you need to build a mechanize script.  Just go to the site you want
to "mechanize" and save each page as you go.  Then run mech-dump against
the local html files and you save a TON of time :)

> allow drag & drop gui building in tk?

Yes, there is -- it's in Komodo (http://www.activestate.com/Products/Komodo/).

You drag-and-drop widgets on a form, and it produces Perl/Tk code.

Komodo and Visual Perl have a lot of overlap -- Perl debugger,
code completion, syntax checking and colorizing, code folding,
PDK integration.

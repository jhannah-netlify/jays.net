[. . .]

> DB<3> x grep { /DAY/ } keys %hash
>    0  'DAYTIME'
>    1  'DAYLIGHT'

[. . .]

> DB<4> x map { $_ => $hash{$_} } grep { /DAY/ } keys %hash
>    0  'DAYTIME'
>    1  3
>    2  'DAYLIGHT'
>    3  2

I can't believe I never thought about how useful map and grep would be
in the debugger.

> On Sep 28, 2005, at 11:14 PM, Dave Thacker wrote:
>> The result is now.
>> "Unrecognized character \xC2 at ./auto-run.pl line 83."

FYI, I've copied and pasted from some e-mail clients before.  Microsoft Outlook and other Rich Text mail clients try to put in "typesetting quotes" rather than the simple ASCII quote character.  When those (or other "pretty") characters are pasted into a file for Perl to use, it sees the wrong character and not the simple ASCII character it was expecting.

Try re-typing the line by hand to see if the \xC2 dissapears.

Wow. That was easy.
Something went terribly wrong with Date::Calc on a server I don't have root on. 

So I installed Date::Calc in my personal lib directory...

$ perl -MCPAN -e install
cpan> o conf makepl_arg "LIB=~/lib INSTALLMAN1DIR=~/man/man1 INSTALLMAN3DIR=~/man/man3" 

cpan> o conf commit

cpan> install Date::Calc

$ export PERL5LIB=/home/jhannah/lib
poof! My own personal Date::Calc works great. I'm surprised I've never been forced to do that before, and am equally surprised at how easy it was. :)

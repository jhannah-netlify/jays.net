Since the tests fail, the install would fail. Bummer.
I am not a C programmer, so this stuff usually makes no sense to me at all.
I googled around a bit and found that apparently this has something to do with the fact that my /tmp directory is mounted as noexec (/etc/fstab). So I switch CPAN/Config.pm to tell it to build everything in /root instead of /tmp (so it uses /root/.cpan instead of /tmp/.cpan). 

Voila, everything is peachy again.

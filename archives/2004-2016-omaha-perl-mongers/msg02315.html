I was wondering something though.  Which one would you do?:

sub goofy_a {
    my $sql = <<'EOF';
        SELECT Foo
        FROM Bar
        WHERE Foo IS NOT NULL
EOF
    return $sql;
}

or

sub goofy_b {
    my $sql = q{
        SELECT Foo
        FROM Bar
        WHERE Foo IS NOT NULL
    };
    return $sql;
}
I like goffy_b because I don't have to have the EOF terminator in column 1 but I think PerCritic complains about using q{} over multiple lines.

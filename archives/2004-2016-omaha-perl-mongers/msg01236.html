No need, I did it here:
$ cat -n TestData

     2  is a
     3  test again||||This is the last test

$ cat -n ParseTestData.pl
     1  #!perl
     2  $/ = '||||';
     3  printf ("Got a line: %s\n",$_) while (<>) ;

$ perl ParseTestData.pl < TestData

Got a line: This is a second test||||
Got a line: This
